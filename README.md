# Inicio

Te damos la bienvenida a nuestra página de documentación oficial del servidor. Nuestro objetivo es ayudar tanto a **nuevos jugadores** como a los **más experimentados** dentro del servidor mediante un lugar con **toda la información necesaria**. Para ello, contamos con este manual que facilitará tanto el **rol diario**, como **dudas**, **reportes**, **definiciones** y mucho **más**.

#### Qué encontrarás aquí

Dentro de la documentación encontrarás numerosas páginas organizadas incluyendo:

* **Normativas:** reglamentos y regulaciones para tener un rol divertido y equilibrado.
* **Guías:** donde podrás encontrar trucos para rolear e información del servidor.
* **Actualizaciones:** registros de las actualizaciones mayores del servidor.
* **Mecánicas y mapeados:** información sobre las áreas del mapa y mecánicas de rol.
* **Artículos:** donde tratamos temas importantes de la comunidad o de interés de rol.
