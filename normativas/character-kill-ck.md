# Character Kill (CK)

Al perder el **100**% de tu **vida**, también puedes elegir **morir** IC y **perder todo tu progreso** en ese personaje para empezar una **nueva historia**. Esto se llama **CK**.

Al hacerte **CK**, pierdes **todo lo relacionado a ese personaje** (**Inventario**, **dinero**, **recuerdos IC**, **rangos** en facciones, **etc**…). También existe el **AutoCK**, que es **suicidarte** IC y el **CKAdmin**, que es cuando se te **penaliza** con un **CK** por **razones administrativas**, como no **valorar tu vida** en un rol de **extrema importancia** o **numerosas** veces.

{% hint style="success" icon="crown" %}
Para ejercer un CKAdmin se necesitará el permiso de **fundación**.
{% endhint %}
