# Alertas

Para regular el **FairPlay** (FRP) en nuestro servidor, usamos un sistema de **alertas**, que establecen una serie de normas sobre las armas que pueden utilizar los jugadores en base a la actividad tanto delictiva como policial. Las alertas actuales son:

* **Verde:** Baja criminalidad, los secuestros o robos no son frecuentes
* **Naranja:** Varios secuestros activos, criminalidad en aumento, robos comunes
* **Roja:** Miembros de cuerpos de seguridad de bajo rango secuestrados, alta criminalidad.
* **Crítica:** Miembros de cuerpos de s. secuestrados, numerosos tiroteos, alta criminalidad.

{% hint style="success" icon="user" %}
**Todos los jugadores deben cumplir con las regulaciones que las alertas establecen.**
{% endhint %}
